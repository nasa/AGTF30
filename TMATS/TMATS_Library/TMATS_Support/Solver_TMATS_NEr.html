T-MATS: Normalized Error Library Block

Purpose

This block can be used to determine the normalized error between two inputs. This is useful for quickly creating error terms that can be used with T-MATS solvers.


Background

To compute the normalized error, this block first determines the difference between the two input values, and then normalizes the result to the first input. This process is summarized by the equation below. $$ NEr = \frac{X1-X2}{X1}$$

Instructions

To use this block:

  • Connect the two input values to the corresponding places on the block.
    • Ensure that the value to which the error will be normalized is input as X1.
  • Connect the output to the next block in your system.



Normalized Error Inputs
Input Description
X1Input 1
X2Input 2


Normalized Error Outputs
Output Description
NErNormalized Error


Normalized Error Mask Variables

No mask variables are needed for this block, as all calculations are based on the input values.

Potential Errors

When using this block, an error will occur if input one is set to zero.
