T-MATS: Outer Loop Integrator Library Block

Purpose

This block is used to provide continuous integration with respect to time of the input value. It is meant to be used in the simulation's outer, time-based loop.


Background

To compute the integrated signal, this block utilizes a standard Simulink integrator block, which outputs the input value integrated with respect to time. The initial conditions for the integration are specified by double-clicking the block and editing the value of the Model Outer Loop Initial Conditions parameter.

$$ X =\int\dot{X}$$
Instructions

To use this block:

  • Connect the input signal to the place labeled Xdot on the block.
  • Connect the output signal to the next block in the outer loop of the simulation.
  • Double click the block and specify the initial conditions.
