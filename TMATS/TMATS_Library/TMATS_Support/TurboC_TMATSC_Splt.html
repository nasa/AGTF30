T-MATS: Splitter (Cantera) Library Block

Purpose

This block is used to simulate the performance of a splitter using basic flow fluid dynamic equations and T-MATS Cantera functions.


Background

To compute the output flow parameters, this block utilizes T-MATS Cantera functions (flowcopy), then adjusts the primary and secondary flow magnitude according to the following equations:

For the primary (main/core) flow path: $$ W_p = \frac{W}{1+BPR} $$

For the secondary (bypass) flow path: $$ W_s = \frac{W*BPR}{1+BPR} $$

Unless adjusted by one of the previous equations, all of the other flow variables remain constant across this block.


Instructions

To use this block:

  • Ensure that a Cantera object exists in the workspace as a global variable named 'fs' and contains all of the species defined in the file 'canteraload.m'.
      Note:This can be created with the Cantera 'importPhase' command.
  • Connect the flow input and bypass ratio inputs to the corresponding places on the block.
  • Connect the outputs to the next blocks in your simulation.

Splitter (Cantera) Mask Variables

No mask variables are needed for this block, since all of the calculations depend only on the inputs.



Potential Errors

When using this block, an error will occur if a Cantera object does not exist in the workspace as a global variable named 'fs'. An error will also occur if this variable does not contain all of the species defined in 'canteraload.m'.
