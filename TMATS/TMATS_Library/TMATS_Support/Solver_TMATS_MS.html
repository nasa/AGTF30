T-MATS: Model Source Library Block

Purpose

This block is used to set up the interface to change simulation variables dynamically.


Background

To determine the output of this block, the user must enter an input vector and a time vector. The time vector is used to allocate when each element of the input vector occurs. Linear interpolation of the input values is then performed to determine the corresponding value at the time that this block is used within the simulation.


Instructions

To use this block:

  • Connect the output to the next block in your simulation.
  • Double click the block and...
    1. Specify the input vector. Note that this parameter should be a function of time.
    2. Specify the time vector.



Model Source Inputs

No direct inputs are required for this block, as the needed inputs are specified by the mask variables.
