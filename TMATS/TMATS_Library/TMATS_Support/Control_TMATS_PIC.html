T-MATS: Simple PI Controller Library Block

Purpose

This block can be used to simulate the effects of a simple proportional-integral (PI) controller.


Background

To simulate a PI controller, this block utilizes the equation below. $$U = E*K_p + K_i*\int{E}$$ in which $$E = R - Y $$ In the above equations, Kp is the proportional gain, Ki is the integrator gain, E is the error as a function of time, R is the input demand, Y is the input to be controlled, and U is output effector demand value. These equations are used to compute the controller output while eliminating any offset of the signal.


Instructions

To use this block:

  • Connect the input demand and input to be controlled to the correct places on the block
  • Connect the output to the next block in your simulation.
  • Double click the block and input the values of the proportional gain, integrator gain, and the initial condition.



Simple PI Controller Inputs
Input Description
Input_demand R - input demanded value
Input_sensed Y - input of plant output to be controlled


Simple PI Controller Outputs
Output Description
Effector Demand U - output to plant Effector
