T-MATS: Data Output Library Block

Purpose

This block is used to save the input variable to a file at each time step. However, it is important to note that the previously saved value is overwritten each time.


Background

To save the input variable, this block utilizes an additional script that saves the value to the specified file. The file is specified by double clicking the block and entering the name of the file.


Instructions

To use this block:

  • Connect the input variable to the corresponding place on the block.
  • Double click the block and specify the file to which the variable should be saved.



Data Output Inputs
Input Description
VarVariable to be saved


Data Output Outputs

No direct outputs are required for this block since the input value is saved to a file.


Output Mask Variables
Mask Variable Description
FVar_MFile name for variable output
