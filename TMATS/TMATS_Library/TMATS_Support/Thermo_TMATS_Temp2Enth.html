T-MATS: Temp2Enth Library Block

Purpose

This block can be used to emperically calculate enthalpy (h) from temperature (T) and fuel to air ratio (FAR). The enthalpy calculated will be total or static, depending on the input of either total temperature (Tt) or static temperature (Ts), respectively.


Background

To compute the enthalpy, this block utilizes a table-lookup method based on data from standard thermodynamic tables for a combination of air and combusted hydrocarbons. The calculations are made from the built-in curve fit data, using the basic equation of: $$h = \frac{h_f*n_{f,p}+h_a*n_{a,p}}{n_{f,p}+n_{a,p}}$$ in which $$h_a = d_{ha,i}*\Delta T^3 + c_{ha,i}*\Delta T^2 + b_{ha,i}*\Delta T + a_{ha,i} $$ and $$h_f = d_{hf,i}*\Delta T^3 + c_{hf,i}*\Delta T^2 + b_{hf,i}*\Delta T + a_{hf,i} $$
In the above equations, ΔT is the difference between the given temperature value and an index value that is obtained from a data array as a function of the integer value of one percent of the assigned gas temperature.

Additionally, the variables a, b, c, and d are arrays of cubic spline-fit coefficients, based on the data tables.


Instructions

To use this block:

  • Connect the temperature and fuel to air ratio inputs to the corresponding places on the block.
  • Connect the enthalpy output to the next block in your simulation.

Temp2Enth Outputs
Output Description

Temp2Enth Mask Variables

No mask variables are needed for this block, as all calculations are based on the input values.
