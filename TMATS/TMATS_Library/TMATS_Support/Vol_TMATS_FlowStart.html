T-MATS: Flow Start Library Block

Purpose

This block simulates the initialization of fluid properties through a lookup table based on total pressure and temperature.


Background

This block simulates the initialization of fluid properties through a lookup table based on total pressure and temperature.


Instructions

To use this block:

  • Connect the inputs of; Gas path char. bus.
  • Connect the outputs of downstream flow properties.
  • Click on the mask to provide details of the two flow components.



Flow Start Inputs
Input Description
W Mass flow [pps]
Tt Total temperature \( [R^\circ]\)
Pt Mass flow [psi]


Flow Start Outputs
Output Description
FlowOutProperties of inflow 5x1 bus consisting of:
W - mass flow rate \( [\frac{lbm}{sec}]\)
ht - enthalp [\( \frac{BTU}{lbm}\)]
Tt - temperature [Rankine]
Pt - pressure \([\frac{lb}{in^3}]\)
FAR - fuel to air ratio [N/A]
GasPropsProperties 14x1 bus of:
rho - Density
u - Internal energy
s - Entropy
mu - Dynamic viscosity
Tt - Total temperature
Cv - Specific heat with constant volume
k - Specific heat ratio
Ts - Static temperature
hs - Static enthalpy
ht - Total enthlapy
Cp - Specific heat with constant pressure
R - Gas constant
gamma - Specific heat ratio


Flow Start Mask Variables
Mask Variable Description
TtVec_M Total temperature vector
PtVec_M Total pressure vector
htVec_M Enthalpy vector
sVec_M Entropy vector
htArray_M Enthlapy table as function of temperature and pressure
TtArray_M Total temperature table as function of enthalpy and pressure
sArray_M Entropy table as function of enthalpy and pressure
rhoArray_M Density table as function of enthalpy and pressure
muArray_M Dynamic viscosity table as function of enthalpy and pressure
CpArray_M Specific heat with constant pressure table as function of enthalpy and pressure
CvArray_M Specific heat with constant volume table as function of enthalpy and pressure
uArray_M Internal energy table as function of enthalpy and pressure
Rval_M Gas Constant
