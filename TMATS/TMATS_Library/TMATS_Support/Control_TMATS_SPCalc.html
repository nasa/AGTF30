T-MATS: Set Point Calculation Library Block

Purpose

This block can be used to determine a set point, either by utilizing a table-lookup method or by a manual input.


Background

Based on the External Request and the Determiner Signal, this block uses a table-lookup to compute the set point. This process can be overridden by entering a manual set point.

Instructions

There are two options for using this block:

  1. Use the table-lookup method
  2. Enter a set point manually
Whichever method you choose to use, remember that you need to:
  • Connect the inputs to the corresponding places on the block.
  • Connect the output to the next block in your simulation.
  • Double click the block and adjust the vector/matrix values, if necessary.
More information on the specifics of what should be entered can be found in the tables below.



Set Point Calculation Inputs
Input Description
External Request External system request for table lookup
Determiner Signal Determiner signal for table lookup
Manual_En Enable manual input: 1- manual input active, 0- use value based on exernal request
Manual Setpoint Manual Setpoint: Setpoint used when manual mode is enabled


Set Point Calculation Outputs
