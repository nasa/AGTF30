T-MATS: SFC Calculation Library Block

Purpose

This block can be used to calculate the specific fuel consumption (SFC) and net thrust.


Background

To compute the net thrust and SFC, this block utilizes the following equations: $$Fnet = Fg - Fram $$
$$SFC = \frac{Wf*3600}{Fg-Fram} = \frac{Wf*3600}{Fnet}$$ In the above equations, Fg is the gross thrust, Fram is the RAM thrust, and Wf is the fuel flow. The constant 3600 in the equation for SFC is used to convert the fuel flow units from pounds per second to pounds per hour.

Instructions

To use this block:

  • Connect the fuel flow and force inputs to the corresponding places on the block.
  • Connect the outputs to the next block in your simulation.



SFC Calculation Inputs
Input Description
WfFuel Flow [pps]
FgGross Thrust [lbf]
FramRAM Thrust [lbf]


SFC Calculation Outputs
Output Description
SFCSpecific Fuel Consumption [(lbm/hr)/lbf]
FnetNet Thrust [lbf]


SFC Calculation Mask Variables

No mask variables are needed for this block, since all of the calculations depend only on the inputs.

Potential Errors

When using this block, an error will occur when computing the specific fuel consumption if Fnet is determined to be zero. To avoid this error, ensure that Fg is not equal to Fram.
