T-MATS: Corrected Speed Library Block

Purpose

This block can be used to calculate the corrected speed.


Background

To compute the corrected speed, this block utilizes the equations below. $$ Nc = \frac{N}{\sqrt{\theta}}$$ in which $$\theta = \frac{Tt}{Tstd}$$
It is important to note that T-MATS assumes the standard day temperature (Tstd) to be 518.67 degR.


Instructions

To use this block:

  • Connect the speed and total temperature to the corresponding input places on the block.
  • Connect the output corrected speed to the next block in your simulation.
  • Adjust the standard day temperature by double clicking the block.



Corrected Speed Inputs
Input Description
NShaft speed
TtTotal Temperature [degR]


Corrected Speed Outputs
Output Description
NcCorrected shaft speed


Corrected Speed Mask Variables
Mask Variable Description
C_TSTD_MStandard Day Temperature [degR]


Potential Errors

When using this block, an error will appear if Tt is set to zero. To avoid this error, ensure that the total temperature input is not equal to zero.
