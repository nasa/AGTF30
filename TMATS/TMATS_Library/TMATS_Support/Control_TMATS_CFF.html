T-MATS: Corrected Fuel Flow Library Block

Purpose

This block can be used to calculate the corrected fuel flow.

Background

To compute the corrected fuel flow, this block utilizes the following equations: $$ Wfc = \frac{Wf}{\delta * \sqrt{\theta}}$$ in which $$\delta = \frac{Pt}{Pstd}$$ and $$\theta = \frac{Tt}{Tstd}$$
It is important to note that T-MATS assumes standard day temperature (Tstd) and pressure (Pstd) to be 518.67 degR and 14.70 psia, respectively.


Instructions

To use this block:

  • Connect the fuel flow, total temperature, and total pressure to the corresponding input places on the block.
  • Connect the output corrected fuel flow to the next block in your simulation.
  • Adjust the standard day values by double clicking the block.

Potential Errors

When using this block, an error will appear if Pt or Tt is set to zero. To avoid this error, ensure that both the total pressure and total temperature inputs are not equal to zero.
