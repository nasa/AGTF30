T-MATS: Splitter Library Block

Purpose

This block can be used to simulate the performance of a splitter using basic fluid dynamic equations.


Background

Across this block, the temperature, enthalpy, pressure, and fuel to air ratio are unchanged. However, the flow (W) is altered according to the follow equations, in which BPR is the bypass ratio. The BPR is tyically a system independent and is derived from an iterative solver.

For the bypass flow path: $$ W_{BFP} = \frac{W*BPR}{1+BPR} $$

For the main flow path: $$W_{MFP} = \frac{W}{1+BPR} $$


Instructions

To use this block:

  • Connect the main flow path gas characteristics and the bypass ratio inputs to the corresponding places on the block.
  • Connect the outputs to the next block in your simulation.



Splitter Inputs
Input Description
MFPGasPthCharInGas Path Characteristics of the Main Flow Path, 5x1 bus/vector consisting of:
- W - Gas path flow [pps]
- ht - Enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]
BPRBypass Ratio
An input of 0 indicates no bypass flow


Splitter Outputs
Output Description
BypGasPthCharOutGas Path Characteristics of the Bypass, 5x1 bus/vector consisting of:
- W - Gas path flow [pps]
- ht - Enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]
MFPGasPthCharOutGas Path Characteristics of the Main Flow Path, 5x1 bus/vector consisting of:
- W - Gas path flow [pps]
- ht - Enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]


Splitter Mask Variables

No mask variables are needed for this block, since all of the calculations depend only on the inputs.
