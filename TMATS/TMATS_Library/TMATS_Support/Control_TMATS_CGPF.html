T-MATS: Corrected Gas Path Flow Library Block

Purpose

This block can be used to calculate the corrected flow.


Background

To compute the corrected flow, this block utilizes the following equations: $$Wc = \frac{W * \sqrt{\theta}}{\delta}$$ in which $$\delta = \frac{Pt}{Pstd}$$ and $$\theta = \frac{Tt}{Tstd}$$
It is important to note that T-MATS assumes standard day temperature (Tstd) and pressure (Pstd) to be 518.67 degR and 14.70 psia, respectively.


Instructions

To use this block:

  • Connect the gas path flow, total temperature, and total pressure to the corresponding input places on the block.
  • Connect the output corrected flow to the next block in your simulation.
  • Adjust the standard day values by double clicking the block.

Potential Errors

When using this block, an error will appear if Pt is set to zero. To avoid this error, ensure that the total pressure input is not equal to zero.
