T-MATS: Duct Library Block

Purpose

This block simulates the performance of a duct.


Background

To compute flow properties, this block utilizes basic fluid dynamic equations and properties. Across the duct, the Total Temperature, Total Enthalpy, Fuel to Air Ratio, and Gas Path Flow are unchanged, however the Total Pressure changes according to the following equation:
$$ Pt_2 = (1-\Delta P)*Pt_1 $$ where ΔP is the fraction of total pressure lost, as specified by the user in the mask variable dP_M based on the properties of the duct. As a reference, in the Steady State Gas Turbine example this value is equal to 0.01.


Instructions

To use this block, connect the core gas path across the Duct block and set the value of the fraction of total pressue lost, dP_M. This is done by:

  • Connecting the GasPthCharOut of the previous block to the GasPthCharIn of the Duct block.
  • Connecting the GasPthCharOut of the Duct block to the GasPthCharIn of the next block.
  • Double-clicking the Duct block and setting the dP_M value for the duct being used.



Duct Inputs
Input Description
GasPthCharInGas Path Characteristics Input, 5x1 bus/vector consisting of:
- W - Gas Path Flow [pps]
- ht - Total Enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]


Duct Outputs
Output Description
GasPthCharOutGas Path Characteristics Output, 5x1 bus/vector consisting of:
- W - Gas Path Flow [pps]
- ht - Total Enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]


Duct Mask Variables
Mask Variable Description
dP_M Delta Pressue [frac lost]
