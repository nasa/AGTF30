T-MATS: Flow Path Characteristics Library Block

Purpose

This block creates a T-MATS GasPathChar bus based on direct flow characteristic inputs.


Background

To compute the flow characteristics, mass flow, total temperature, total pressure, and fuel to air ratio are input. The output is a T-MATS GasPathChar bus that may be wired into any other T-MATS block.


Instructions

To use this block:

  • Connect the flow, total temperature, total pressure, and fuel to air ratio \ to the corresponding places on the block.
  • Connect the output to the next block in your simulation.



Flow Path Characteristics Inputs
Input Description
WGas path flow[pps]
TtTotal Temperature [degR]
PtTotal Pressure [psia]
FARCombusted Fuel to Air Ratio [frac]


Flow Path Characteristics Outputs
Output Description
GasPthCharOutGas Path Characteristics Output, 5x1 bus/vector consisting of:
- W - Gas path flow [pps]
- ht - Enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]


Flow Path Characteristics Mask Variables

No mask variables are needed for this block, since all of the calculations depend only on the inputs.
