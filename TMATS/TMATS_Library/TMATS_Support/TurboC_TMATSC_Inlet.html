T-MATS: Inlet (Cantera) Library Block

Purpose

This block is used to simulate the performance of an inlet using basic fluid dynamic equations and properties and T-MATS Cantera functions.


Background

To compute the output flow parameters, this block utilizes begins by determining the output pressure from the input pressure and the user-specified recovery, as seen in the equation below. $$Pt_{Out} = Pt_{In}*Recovery$$ Other exit conditions are set according to the incoming enthalpy and exit pressure, using T-MATS Cantera functions.


Instructions

To use this block:

  • Ensure that a Cantera object exists in the workspace as a global variable named 'fs' and contains all of the species defined in the file 'canteraload.m'.
      Note:This can be created with the Cantera 'importPhase' command.
  • Connect the flow input to the corresponding place on the block.
  • Connect the outputs to the next blocks in your simulation.
  • Double click the block and specify the inlet recovery as a fraction.

Potential Errors

When using this block, an error will occur if a Cantera object does not exist in the workspace as a global variable named 'fs'. An error will also occur if this variable does not contain all of the species defined in 'canteraload.m'.
