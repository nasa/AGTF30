T-MATS: Shaft Library Block

Purpose

This block can be used to simulate the performance of a shaft using basic dynamic equations and properties.


Background

To compute the acceleration (Ndot), this block takes the sum of the input torque elements and uses the equation below. $$ Ndot = \frac{60*\sum{Torque}}{2\pi*\textit{Shaft_Inertia_M}}$$ Since the input torque can be a vector, the summation is used to add all of the components together. The constants in the above equation are used to convert the input units to the proper output units for the shaft's acceleration. Generally negative torque is used for energy being taken from the shaft and positive torque is for energy put into the shaft.

It should be noted that the Nmech input is simply passed through to the output and is needed only for understandability purposes. Calculation of Nmech is completed with the integration of shaft output Ndot using an iterative solver.


Instructions

To use this block:

  • Connect the torque and mechanical shaft speed to the corresponding places on the block.
  • Connect the outputs to the next block in your simulation.
  • Double click the block and specify the shaft inertia.



Shaft Inputs
Input Description
TorqueInput torque from torque source or sink [lbf*f]
NmechInMechanical shaft speed[rpm].


Shaft Outputs
Output Description
NmechOutMechanical shaft speed[rpm], equal to the input shaft speed.
NdotMechanical shaft speed acceleratoin [rpm/sec]


Shaft Mask Variables
Mask Variable Description
Shaft_Inertia_M Shaft Inertia Term [slugs*ft^2]


Potential Errors

When using this block, an error will occur if the shaft inertia is set to zero.
