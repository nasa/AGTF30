T-MATS: Gearbox Library Block

Purpose

This block can be used to simulate the performance of a gearbox.

Background

To compute the shaft speed and torque, this block utilizes basic gear properties, such as: $$\textit{R_M} = \frac{NmechA}{NmechB} =\frac{TrqB}{TrqA} $$ in which R_M is the gear ratio and A and B denote sides of the gear box. The above equation is re-written to solve for the outputs, as seen below: $$ NmechB = \frac{NmechA}{R_M}$$ and $$ TorqA = \frac{TorqB}{\textit{Eff_M*R_M}}$$ in which Eff_M is the efficiency of the gearbox.

Instructions

To use this block:

  • Connect the input and output torques and shaft speeds to the corresponding places on the block.
  • Double click on the block and specify the gear ratio and the gear efficiency.



Gearbox Inputs
Input Description
TorqBTorque at gearbox side B [lbf*ft]
NmechAMechanical shaft speed at gearbox side A [rpm]


Gearbox Outputs
Output Description
NmechBMechanical shaft speed at gearbox side B [rpm]
TorqATorque at gearbox side A [lbf*ft]


Gearbox Mask Variables
Mask Variable Description
R_MGear Ratio [frac]
Eff_MGear box efficiency from A to B [frac]


Potential Errors

When using this block, an error will occur if either Eff_M or R_M is set to zero.
