T-MATS: Color Guide

Background

To aid in simulation setup, the inputs and outputs of T-MATS blocks are color coded according to the table below.



Color Guide
Color Description
Blue Gas path information containing flow, enthalpy, total temperature, total pressure, and flow composition (typically fuel to air ratio (FAR))
Dark Green Independent variable
Green Dependent variable
Red Independent variable that is obtained through integration in a dynamic simulation, but may be solved for by the solver in a steady state simulation (e.g. shaft speed)
Magenta Dependent variable that is integrated in a dynamic simulation, but may be used as a dependent variable in the iterative solver in the steady state simulation (e.g. Ndot)
Pink Contributor to the calculation of a dependent variable (e.g. torque)
Yellow Effector command (e.g. fuel flow), not typically used by the solver
