T-MATS: Independent Intial Condition Block
This block translates a fracional input to the final value based on an initial guess. This is typically used to scale or translate outputs from the Iterative solver from a simple fraction to the final value used. The purpose of performing this action is to enhance plant readability by placing the initial guesses for the independents into the plant. In addition performing these transformations allow the user to force all the iterative solver values to a similar format.
