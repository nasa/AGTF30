T-MATS: Inlet Library Block

Purpose

This block can be used to simulate the performance of an inlet using basic fluid dynamic equations and properties.


Background

To compute the gas path characteristics, this block utilizes an emperical approach to determine inlet effects on flow. Across the inlet, pressure changes are developed according to the following equation: $$ \textit{PtOut} = \textit{PtIn} * \textit{eRambase_M}*\textit{T_eRamtbl_M}\left(\frac{PtIn}{Pamb}\right) $$ in which eRambase_M is the base fraction of pressure lost, as specified by the user and T_eRamtbl_M is an off nominal fractional pressure loss determined by a table-lookup using the values of PtIn and Pamb. Additionally, Changes in temperature are considered negligible.

Instructions

To use this block:

  • Specify the ambient pressure by connecting it to the corresponding place on the block.
  • Connect the gas path across the block by inputting the gas path characteristics from the previous block to the input of this block and by connecting the output of this block to the input of the next block in your simulation.
  • Specify the value of the base fractional pressure lost across the inlet by double clicking the block.



Inlet Inputs
Input Description
GasPthCharInGas Path Characteristics Input, 5x1 bus/vector consisting of:
- W - Gas path flow [pps]
- ht - Enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]
PambAmbient pressure [psia]


Inlet Outputs
Output Description
GasPthCharOutGas Path Characteristics Output, 5x1 bus/vector consisting of:
- W - Gas path flow [pps]
- ht - Enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FAR - Combusted Fuel to Air Ratio [frac]


Inlet Mask Variables
Mask Variable Description
eRambase_MBase delta pressure [frac]
X_eRamVec_MPt/Pamb Vector [nx1]
T_eRamtbl_Mdelta pressure based on Pt/Pamb [frac][nx1]


Potential Errors

When using this block, an error will occur if Pamb is set to zero. To avoid this error, ensure that the input ambient pressure is nonzero.
