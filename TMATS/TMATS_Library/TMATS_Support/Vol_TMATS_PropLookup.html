T-MATS: Property Lookup Library Block

Purpose

This block is used to model the thermodynamic properties of a mass flow based on table lookups.


Background

This block is used to model the thermodynamic properties of a mass flow based on table lookups. This table lookup is based on a total pressure and a defined property value set within the mask. Note: units are defined as the user describes the tables.


Instructions

To use this block:

  • Connect the inputs of; property value and total pressure.
  • Connect the outputs of; downstream flow properties and convergence value.
  • Click on the mask to provide inputs the property case, initial guesses, and property tables.



Prop Lookup Inputs
Input Description
Property Property
Pressure total pressure \([\frac{lbm}{in^3}]\)


Prop Lookup Outputs
Output Description
PropsProperties 14x1 bus of:
rho - Density
u - Internal energy
s - Entropy
mu - Dynamic viscosity
Tt - Total temperature
Cv - Specific heat with constant volume
k - Specific heat ratio
Ts - Static temperature
hs - Static enthalpy
ht - Total enthlapy
Cp - Specific heat with constant pressure
R - Gas constant
gamma - Specific heat ratio
Pt Total pressure
