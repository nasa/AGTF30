T-MATS: Simulink Tools

Background

T-MATS tools have been created to automate certain tasks in T-MATS. T-MATS Simulink tools can be accessed from a Simlink diagram via the menu Diagram > T-MATS. For T-MATS tools to work properly, the file Menu_customization_TMATS.m located within the TMATS_Tools folder in the T-MATS directory must be run.

To automate loading, add the lines below to the startup.m script and verify the TMATS_Tools folder is on the permanent path list.

load_simulink
Menu_customization_TMATS

Other T-MATS tools include:
1) Gas Table Builder
2) NPSS to TMATS Conversion Tool
(see \TMATS_Tools folder for access)
